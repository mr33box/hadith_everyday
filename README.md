# 📱 Daily Hadith

> "Hadith every day, keep the shaitan away"

Daily Hadith is a simple and beautiful mobile app that delivers an authentic hadith every day and automatically sets it as your phone wallpaper.

---

## ✨ Features

- 📖 **Daily Hadith**
  - Get a new hadith every day (or custom time)
  
- 🖼️ **Auto Wallpaper**
  - Automatically set hadith as your home & lock screen wallpaper

- 🎨 **Custom Design**
  - Fully customize:
    - Background colors
    - Text colors
    - Font size
    - Layout & alignment

- 🌙 **Light & Dark Mode**
  - Clean UI with both themes

- ❤️ **Favorites**
  - Save your favorite hadiths

- 📚 **History**
  - View all previously generated hadiths

- 🌐 **Multi-language**
  - Arabic 🇸🇦
  - English 🇬🇧
  - Fully synced across UI + wallpaper

## ⚙️ How It Works

1. App fetches a hadith from API
2. Generates a beautiful image
3. Applies it as wallpaper automatically
4. Updates based on your chosen interval

---

## 🎯 Why This App?

- Helps you **remember hadith daily**
- Turns your phone into a **source of reminder**
- Simple, lightweight, and distraction-free
